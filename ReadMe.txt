# Portfolio Optimization with Machine Learning

This project implements a two-stage framework combining machine learning-based return prediction with portfolio optimization.

## Models
- Random Forest (baseline)
- XGBoost
- LSTM

## Features
- Lagged returns
- Technical indicators
- Market features

## How to Run
1. Install required libraries
2. Run preprocessing scripts
3. Train models
4. Run portfolio optimization
